# intelligent_reporting

Contains the main modules of the project for **data profiling and reporting**:


- `loading/` – Data loading, saving, and connectors.  
- `profiling/` – Schema inference, statistics, and data profiling logic.  
- `reporting/` – Text summaries, visuals, and dashboards.  
- `exporters/` – Exporting reports to any format and/or send them by mail.
- `agents/` – Agents for data analysis and reporting.
